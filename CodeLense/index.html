Metrics Explained

Cyclomatic Complexity

Counts decision points in code (if/else, loops, switches). Higher = harder to test. Threshold: ≤10 per function.

Cognitive Complexity

Measures how hard code is to read. Penalizes nesting and complex control flow. Threshold: ≤15.

Function Length

Lines of code per function. Long functions are harder to maintain and test. Threshold: ≤30 lines.

Maintainability Index

Weighted composite: 0–100. A=80+, B=60–79, C=40–59, D=20–39, F=<20. Higher is better.
