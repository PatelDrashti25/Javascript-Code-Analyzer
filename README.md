 # JavaScript Code Quality Analyzer

## MSc Project

This project presents a hybrid intelligent system for analyzing JavaScript code quality using:

- Static analysis
- Code metrics
- Heuristic ML-based detection
- AI-inspired suggestions

## Features
- Cyclomatic & Cognitive Complexity
- Code Smell Detection
- Maintainability Index
- AST Analysis
- Code Duplication Detection
- Interactive UI

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
Open index.html in browser
